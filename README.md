# pmcram-legal

Public legal pages for the **PM Cram (PMP Prep)** app — Privacy Policy and Terms of Use /
EULA — hosted with **GitHub Pages** so the App Store and Google Play (and the app itself)
can link to them.

| Governing law | England and Wales |
| Effective date | 13 July 2026 |

Update the **effective date** if you publish on a different day.

## Publish with GitHub Pages

1. Create a new **public** GitHub repo (e.g. `pmcram-legal`) and push these files.
2. Repo → **Settings → Pages** → Source: **Deploy from a branch** → Branch: `main` → `/ (root)`
   → Save.
3. After a minute your pages are live at:
   - `https://forge-deck.github.io/pmcram-legal/privacy/`
   - `https://forge-deck.github.io/pmcram-legal/terms/`

## Where the URLs go

- **App Store Connect** → your app → **App Privacy** → *Privacy Policy URL* = the `/privacy` URL.
- **Google Play Console** → **Store presence → Store listing** (and **App content → Privacy
  policy**) = the `/privacy` URL.
- **In the app** (optional, recommended): link the Privacy and Terms URLs from the unlock screen
  and Settings.
